# Frontend Design Guidelines

This project follows the frontend-design skill guidelines for creating distinctive, production-grade interfaces.

## Design Direction

- **Purpose**: Modern coffee shop landing page showcasing artisanal coffee experience
- **Tone**: Luxury/refined with warmth and elegance - sophisticated but welcoming
- **Differentiation**: Premium feel with smooth animations and rich visual depth

## Aesthetics Guidelines

### Typography
- Use distinctive fonts that elevate the design
- Pair a refined display font (Playfair Display) with a clean body font (Inter)
- Avoid generic fonts like Arial, Roboto, or default system fonts

### Color & Theme
- Dark theme with warm gold accents (#c9a66b)
- Cohesive color palette using CSS variables
- Dominant colors with sharp accents

### Motion
- Staggered entrance animations on page load
- Scroll-triggered animations for sections
- Smooth hover states with subtle transforms
- Use GSAP for complex animations

### Spatial Composition
- Generous spacing between sections
- Asymmetric layouts where appropriate
- Clear visual hierarchy

### Visual Details
- Glassmorphism header with backdrop blur
- Subtle shadows and depth
- High-quality imagery from Unsplash
- Smooth transitions throughout

## Implementation

- Production-grade, functional code
- Visually striking and memorable
- Cohesive with clear aesthetic point-of-view
- Meticulously refined in every detail

## Avoid
- Generic "AI slop" aesthetics
- Overused fonts (Inter, Roboto, Arial)
- Clichéd color schemes
- Predictable, cookie-cutter layouts
